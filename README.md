# Java_Polyclinic

This is the registry of the polyclinic
Used Java + MySQL

If the project doesn’t work, just edit Run Configuration with this:

--module-path
**Path to project**/lib/javafx-sdk-11.0.2/lib
--add-modules
javafx.controls,javafx.fxml
